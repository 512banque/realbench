# R-025-004: Calibration Module Cold-Chain Transport

Calibration modules may be transported by drone only when the carrier holds a
technical cold-chain endorsement and a technical handling manifest has been
filed before dispatch. If either requirement is missing, the transport is
prohibited.

This rule controls the module-class requirements. It does not by itself waive
lane, time, emergency, or passenger-flight restrictions.

Condition markers: `C-025-cold-chain-endorsement`,
`C-025-technical-manifest`.

Effective date: 2027-03-01.
