# R-025-001: Municipal Aerial Lane Day Operations

A licensed drone carrier may operate in a municipal aerial lane for routine
parcel delivery between 06:00 and 22:00 local time.

This rule does not authorize night operation. If a later rule specifically
addresses night operation, apply the later rule for the night-flight question.

Condition marker: `C-025-licensed-carrier`.

Effective date: 2027-01-01.
